1. The Registry of Errors

BRAH: The Yapping Violation Occurs when comments (~ ~) exceed 1.5x the length of functional code. Fix: Write more code or stop explaining yourself.
BRUH: The Type Betrayal Occurs when you try to perform math on a String without using the Melter. Fix: Use \'var**'\ to convert to a number.
STAP: The Reality Check Occurs when attempting impossible math (like dividing by 0 or imaginary roots). Fix: Follow the laws of physics.
